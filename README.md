# BidCraft Agentic UI

This repository contains a React + TypeScript UI (Vite) port of an internal BidCraft Agent prototype.

Quick start:

- Install dependencies: `npm install`
- Run dev server: `npm run dev`
- Typecheck: `npm run typecheck`
- Build: `npm run build`

Notes:
- Tailwind CSS is used for styling. Configuration is in `tailwind.config.cjs`.
- Icons are from `lucide-react`.
