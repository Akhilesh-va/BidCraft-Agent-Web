# BidCraft Agentic AI — WebApp

This is the BidCraft Agentic AI frontend built with Vite + React + TypeScript + styled-components.

Quick start:

1. cd "BidCraft Agent WebApp"
2. npm install
3. npm run dev

Notes:
- Designed to reflect the mobile UI screenshots provided — a soft card-based layout, rounded elements, and an "agentic" progress timeline.
- Folder layout: `src/components`, `src/pages`, `src/hooks`, `src/utils`, `src/styles`
